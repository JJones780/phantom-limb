Each of these boxes will log something to the console when touched.

If you don't have a console, Firebug Lite is provided.

View source to see how to initialize Phantom Limb

These have event handlers added programmatically:
touchstart (added) touchmove (added) touchend (added)

These have event handlers added inline:
ontouchstart (inline) ontouchmove (inline) ontouchend (inline)
